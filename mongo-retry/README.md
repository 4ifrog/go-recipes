# Retries on MongoDB

An example of how you would keep retrying to connect to MongoDB.

## Requirements 

For this example, we deliberately don't want to connect to the database as we want to build a robust retry strategy. 

## Setup

1. Run the application

   ```bash
   $ go run main.go
   ```
